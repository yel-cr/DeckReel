# DeckReel  
SteamDeckのためのシンプルなスクリーンショット管理ツール。  
  
## できること  
- **SS整理**：SteamDeckで撮影したスクリーンショットをゲームごとに分類、閲覧  
- **Rclone連携**：各種オンラインストレージに一括アップロード  
- **ゲームモード対応**：一度設定を終えれば、その後はゲームモードで使用できます  
  
## 動作環境/必要なもの  
  
### SteamOS  
このツールはSteamDeckでの使用を前提に開発されています。その他の環境での動作は検証されていません。  
  
### Rclone  
https://rclone.org

オンラインストレージへのアップロードに使用します。公式サイトからダウンロードして、管理しやすいところに配置してください。  
  
## 簡単な使い方  
1. rclone configでリモート作成  
2. deckreel.shを実行  
3. 各種パスやリモート名などを設定  
4. Resolveをクリックしてゲーム名を取得  
5. Sync Allをクリックしてアップロード
